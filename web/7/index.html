Simple by design

Per-keyboard layouts

Assign a different input language to each physical keyboard plugged into your machine. It just remembers.

Instant switching

The moment you press a key, GlueyKeys detects which keyboard it came from and switches layout instantly.

Runs quietly

Lives in your system tray. No pop-ups, no interruptions. It stays out of your way until you need it.

Easy setup

Install, pick a layout for each keyboard, done. No drivers, no registry hacks, no restarts needed.

Any keyboard

USB, Bluetooth, built-in laptop keyboard, wireless dongles. If Windows sees it, GlueyKeys can manage it.

Lightweight

Tiny footprint. Minimal CPU and memory usage. Starts with Windows and never slows you down.

Built in the open

GlueyKeys is open source. Browse the code, report issues, or contribute on GitHub.
