Features

Simple by design

Per-keyboard layouts

Assign a different language or layout to each physical keyboard connected to your machine. Each one remembers its own setting.

Instant switching

The moment you start typing on a different keyboard, the layout switches automatically. Zero delay, zero effort.

Runs quietly

Lives in your system tray and stays out of the way. No pop-ups, no interruptions. It just works in the background.

Easy setup

Install, pick a layout for each keyboard, and you are done. Configuration takes less than a minute.

Any keyboard

Works with USB, Bluetooth, wireless dongles, and built-in laptop keyboards. If Windows sees it, GlueyKeys handles it.

Lightweight

Tiny footprint, minimal memory usage, and no bloat. It does one thing and does it well without slowing your system down.

Open source

Built in the open

GlueyKeys is free and open-source software. Browse the code, suggest features, or report bugs — contributions are always welcome.
