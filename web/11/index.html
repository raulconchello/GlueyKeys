Simple by design

GlueyKeys remembers which layout belongs to which keyboard. Plug in and type — no switching, no thinking.

Automatic detection

Detects each physical keyboard the moment you plug it in. No manual setup or device pairing needed.

Per-keyboard layout

Assign a different language or layout to each keyboard. Switch between QWERTY, AZERTY, or any layout seamlessly.

Runs in the tray

Sits quietly in your system tray. Zero distraction, zero overhead. Right-click to configure anytime.

Instant switching

Layout changes the moment you press a key on a different keyboard. No delay, no flicker.

Remembers forever

Your preferences persist across reboots. Set it once and never think about it again.

Free and open source

MIT licensed, built in the open. Inspect the code, file issues, contribute — it's yours.

Built in the open

GlueyKeys is open source and community driven. Browse the code, report bugs, or suggest features — every contribution helps.
