Features

Simple by design

Auto layout switching

Detects which physical keyboard you are typing on and switches the OS layout automatically. No hotkeys, no thinking.

Per-keyboard memory

Remembers the layout you assigned to each keyboard. Plug in any device and it picks up right where you left off.

System-tray quiet

Runs silently in the background. A single tray icon is all you see. No windows, no popups, no distractions.

Instant detection

Identifies the active keyboard on the very first keystroke. The switch happens before you finish the word.

Zero configuration

Install, assign your layouts once, and forget about it. No config files, no complicated settings panels.

Lightweight and safe

Tiny footprint, no admin rights needed, no data leaves your machine. Open source so you can verify every line.

Open source

Built in the open

GlueyKeys is free and open source. Read the code, report bugs, suggest features, or contribute directly.
