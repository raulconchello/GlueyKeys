Simple by design

GlueyKeys detects which physical keyboard you're typing on and instantly switches to the right layout. No menus, no shortcuts, no thinking.

Instant switching

Layouts change the moment you press a key on a different keyboard. Zero delay, zero friction.

Per-keyboard memory

Assign a layout to each physical keyboard once. GlueyKeys remembers every time you reconnect.

System tray app

Lives quietly in the tray. No window to manage, no CPU drain. Just works in the background.

Multi-keyboard aware

Built-in laptop keyboard, external mechanical board, Bluetooth split — handle them all at once.

Easy configuration

Pick a keyboard from the list, choose its layout. That's it. One click to pair, one click to change.

Lightweight

Tiny footprint. No Electron, no runtime dependencies. A native Windows app that starts in milliseconds.

Built in the open

GlueyKeys is free and open-source. Browse the code, report bugs, or suggest features on GitHub.
