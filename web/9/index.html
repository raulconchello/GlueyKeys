Each keyboard,
its own language.

GlueyKeys remembers which layout belongs to which keyboard.

Simple by design

No bloat. No config files. It just works.

Per-keyboard layouts

Assign a different language layout to each physical keyboard. Switch devices, not settings.

Instant detection

GlueyKeys detects the active keyboard the moment you press a key and switches layout in real time.

Runs in the tray

Lightweight system-tray app. Starts with Windows, stays out of your way, uses minimal resources.

No admin required

Installs and runs without elevated privileges. No drivers, no kernel hooks, no risk.

Private by default

Everything stays on your machine. No analytics, no telemetry, no network calls. Ever.

Open source

MIT-licensed. Read every line, fork it, improve it. The code is yours to inspect and extend.

Built in the open

GlueyKeys is free, open-source software. Browse the code, report bugs, or suggest features — contributions are always welcome.
