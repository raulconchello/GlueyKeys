How it works

Simple by design

Per-keyboard layouts

Assign a different language layout to each physical keyboard connected to your PC. No manual switching needed.

Instant switching

Detects keystrokes in real time and changes the layout instantly. No lag, no delay, just seamless typing.

Runs quietly

Lives in the system tray. No pop-ups, no notifications. It works silently in the background so you can focus.

Easy setup

Pick a keyboard, pick a layout, done. The configuration UI takes seconds, not minutes.

Any keyboard

Works with USB, Bluetooth, and wireless keyboards. If Windows sees it, GlueyKeys can manage it.

Lightweight

Minimal memory footprint, near-zero CPU usage. Designed to do one thing and do it well, nothing more.

Open source

Built in the open

GlueyKeys is free and open source. Browse the code, contribute fixes, or fork it and make it your own.
