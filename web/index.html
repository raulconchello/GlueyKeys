How it works

One keyboard, one language. Always.

Assign a specific input language to each physical keyboard. Your laptop stays English, your external stays Spanish. GlueyKeys switches instantly when you type. No manual toggling, no shortcuts to remember.

The problem

Windows keeps changing your keyboard layout

A 10+ year old bug that Microsoft has never fixed. Sound familiar?

"Only for writing this small text, I had to change back 5 times to the layout I am using."
Microsoft Q&A user
"Everything works fine, but after I restart my PC, all of my keyboard settings get reset."
Multilingual Windows user
"I want Colemak and it keeps changing back to QWERTY even if I delete QWERTY from the list!"
Custom layout user
Layout resets on every restart

Your chosen layout reverts to the default after restart, sleep, or login. Every. Single. Time.

Ghost layouts keep reappearing

Windows adds keyboard layouts you never installed. Delete them, they come back after reboot.

No per-keyboard layout support

Spanish laptop + English external keyboard? You have to manually switch every time you move between them.

None of the "fixes" actually work

Registry hacks, PowerShell scripts, obscure settings. Users try 5-10 workarounds and the problem always comes back.

GlueyKeys fixes all of this.

Assign a layout to each keyboard once. It sticks, through restarts, sleep, and updates. No registry hacks. No workarounds. It just works.

Features

Simple by design

Per-keyboard layouts

Assign a different input language to each physical keyboard. Your laptop stays English, your external stays Spanish, automatically.

Instant switching

The moment you press a key, GlueyKeys detects which keyboard sent it and switches the layout in under a millisecond.

Runs quietly

Sits in the system tray and does its job. No pop-ups, no interruptions, no unnecessary notifications.

Easy setup

Pick a keyboard, pick a layout, done. The straightforward interface takes seconds to configure, not minutes.

Any keyboard

Works with any USB or Bluetooth keyboard Windows can see. No special drivers or hardware requirements.

Lightweight

Tiny footprint, minimal memory, no bloat. It starts with Windows and stays out of your way.
