Each keyboard, its own language.

Auto-switches your Windows input language when you type on a different keyboard.

Simple by design

Per-keyboard layouts

Assign a specific language to each physical keyboard. Your laptop gets English, your external keyboard gets Spanish.

Instant switching

The moment you press a key, GlueyKeys detects which keyboard it came from and switches to the right layout.

Runs quietly

Lives in your system tray. No windows, no interruptions. Just works in the background.

Easy setup

A wizard guides you through first-time configuration. Press a key on any keyboard and pick its layout.

Any keyboard

Works with USB, Bluetooth, and built-in laptop keyboards. If Windows sees it, GlueyKeys can map it.

Lightweight

Native Windows app built with WPF. Minimal memory footprint, no background services.

Built in the open

GlueyKeys is free and open source. Found a bug? Want a feature? Contributions are welcome.
