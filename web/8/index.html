Simple by design

Plug in any keyboard. GlueyKeys remembers which layout belongs to which device. No config files. No restarts.

Automatic detection

GlueyKeys identifies each physical keyboard the moment you plug it in -- no manual setup required.

Per-keyboard layouts

Assign a different language or layout to each keyboard. Switch physically, not mentally.

Instant switching

Layout changes happen in under 50 ms. Type on one keyboard, switch to another -- zero lag.

Runs in the tray

Sits quietly in your system tray. Uses minimal resources. Stays out of your way until you need it.

No admin rights

Installs and runs without elevated permissions. No drivers, no kernel hooks, no risk.

Open source

MIT licensed. Read every line of code, fork it, improve it. Built by the community, for the community.

Built in the open

GlueyKeys is free, open-source software. Report bugs, request features, or contribute code -- every issue and pull request makes it better for everyone.
