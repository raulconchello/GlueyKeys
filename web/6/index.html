Simple by design

One job, done right. GlueyKeys stays out of your way and just works.

Per-keyboard layouts

Assign a different input language to each physical keyboard you plug in. Laptop stays English, external goes Spanish.

Instant switching

Layout changes the moment you press a key on a different keyboard. No delays, no manual toggling.

Set it and forget it

Configure once in the system tray. GlueyKeys starts with Windows and runs silently in the background.

Lightweight and safe

No admin rights needed. No network calls. Tiny memory footprint. Just a clean Windows app.

Works with any keyboard

USB, Bluetooth, wireless dongle -- if Windows sees it, GlueyKeys can assign a layout to it.

Open source

MIT licensed. Read every line, fork it, improve it. Built transparently on GitHub for everyone.

Built in the open

GlueyKeys is open source. Browse the code, report bugs, or contribute features.
