Simple by design

GlueyKeys does one thing and does it well.

Per-keyboard layouts

Assign a different language layout to each physical keyboard plugged into your machine. Switch keyboards, not settings.

Instant switching

Layout changes happen the moment you start typing on a different keyboard. No delay, no interruption.

Runs quietly

Lives in the system tray and stays out of your way. No pop-ups, no notifications, just works.

Easy setup

Plug in a keyboard, pick a layout from the tray menu, done. Configuration takes seconds, not minutes.

Any keyboard

Works with any USB or Bluetooth keyboard Windows can see. No special hardware or drivers required.

Lightweight

Minimal resource footprint. Uses almost no CPU or memory so your machine stays fast.

Built in the open

GlueyKeys is open source. Browse the code, report a bug, or suggest a feature—contributions are welcome.
